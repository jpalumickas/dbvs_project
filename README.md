DBVS Project in University
============

DBVS Project written in Java language, which manage fuel consuption in work.

## Usage
  
Clean builds

```sh
ant clean
```
    
Compile Java project

```sh
ant compile
```
    
Make Jar file of Java project

```sh
ant jar
```

Clean, compile, build Jar and run project
```sh
./run
```
